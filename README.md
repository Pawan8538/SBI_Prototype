## 🌟 Key Features

Our prototype extends the core SBI YONO banking functionality with **four cutting-edge AI financial wellness features**:

### 1. Smart Budget Coach 📊
An intelligent dashboard that analyzes spending patterns in real-time. It categorizes expenses (Food, Bills, Lifestyle), alerts you when you are nearing your budget limits, and offers actionable advice to optimize your cash flow.

### 2. Hyper-Personalized Goal-Based Savings Agent 🎯
Reimagines traditional savings by allowing users to set dynamic goals (e.g., buying a laptop, taking a vacation, building an emergency fund). 
- Automatically calculates the exact monthly savings required based on your timeline and target amount.
- Provides dynamic AI insights to keep you on track.

### 3. Digital Financial Twin (Simulation Engine) 🧬
A powerful forecasting tool that allows you to simulate your financial future over the next 5 years. 
- Adjust complex variables like **Monthly SIP Investments, Annual Salary Hikes, Expense Reductions, Simulated Loans, Retirement Age, and Interest Rates**.
- The dynamic Area Chart instantly projects your net worth and compares your simulated outcome against your current trajectory.

### 4. Interactive Voice-Based Banking Assistant 🎙️
A seamless, hands-free navigation and intent engine.
- Instead of manually clicking through menus, users can trigger the floating Voice Assistant overlay.
- Our mock interactive engine processes commands like *"Show my budget"*, *"Create a savings goal"*, or *"Simulate my future net worth"* and automatically navigates the app on your behalf.
- **Auto-Demo Mode:** Click the mic button to watch the assistant automatically type and execute commands based on your current screen!

### 5. ASTRA Core Engine & Unified Balances ⚙️
- **1-Click Sovereign Allocations:** ASTRA intercepts user cash flows and recommends low-risk sovereign investments (like PPF or SGB). Users can execute these investments in one click.
- **Global Balance Synchronization:** The account balance is managed globally. When an investment is confirmed, the exact amount is instantly deducted across the entire application (including YONO Pay and transaction histories).

## 🚀 Running Locally

If you'd like to clone and run the prototype locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pawan8538/SBI_Prototype.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd SBI_Prototype/sbi-astra-prototype
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
5. **Open your browser:** Navigate to `http://localhost:5173`
